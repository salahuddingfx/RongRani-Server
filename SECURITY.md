# Security Policy

## Supported Versions

We actively support and fix security issues on the following versions of RongRani:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of RongRani very seriously. If you find any security vulnerabilities in the code, please do not report them via public GitHub issues. Instead, report them directly to our security team.

Please email all security reports to **info.rongrani@gmail.com** with the following details:

1. **Description of the vulnerability**: What is the bug and the security impact?
2. **Steps to reproduce**: Proof of concept (PoC) scripts, steps, or screenshots to demonstrate the issue.
3. **Suggested mitigation**: If you have a fix or workaround, please include it.

We will acknowledge receipt of your report within **48 hours** and provide a detailed response with next steps within **7 days**. We request that you follow responsible disclosure guidelines and keep details private until a patch is released.
